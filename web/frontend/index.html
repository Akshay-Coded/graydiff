About this result

This project trained a neural surrogate to imitate a NumPy Gray-Scott solver, validated that it reproduces the correct Turing-pattern regimes across a bounded rollout horizon (not indefinitely — see the project notebooks for the honestly-measured stability window), then used backpropagation through the frozen, differentiable surrogate to solve the inverse problem: given a drawn pattern, recover the physics parameters that produce it. Every recovered result shown here is independently re-verified against the real NumPy solver, server-side — never graded only against the surrogate's own output.

Full build notebooks (00–06), the ground-truth recovery experiment with a measured recovery error, and an explicit failure-mode gallery (including cases where this doesn't work well) are in notebooks/ in the project repository.
